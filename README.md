# Flavour Hub
 
A web application built with **ASP.NET MVC 5, Entity Framework, and SQL** to manage recipes and ingredients.
 
## Features
- Add/View/Delete recipes
- Search functionality
- User-friendly UI
 
## Tech Stack
- C#, ASP.NET MVC 5
- Entity Framework 6
- SQL Server
- HTML, CSS, JavaScript
 
## Run the Project
1. Clone the repository
2. Open in Visual Studio
3. Restore NuGet packages
4. Run the application
